## ✨ 新功能

### 核心功能
- 📁 **目录平铺**：递归遍历源目录，将所有文件移动到目标目录的根层级
- 🔄 **移动/复制模式**：支持移动或复制文件，满足不同使用场景
- ⚡ **智能冲突处理**：支持覆盖、跳过、自动重命名三种冲突处理策略
- 🎭 **演习模式**：预览操作结果而不实际执行，确保操作安全
- 🧹 **自动清理**：操作后自动清理空文件夹，保持目录整洁
- ↩️ **撤销操作**：支持撤销上一次的移动操作，提供后悔药
- 📝 **操作日志**：记录所有操作历史，方便追踪和审计
- 📊 **实时反馈**：带进度条的操作反馈，清晰显示执行进度

### 用户界面
- 🎨 **现代化设计**：简洁直观的图形用户界面
- 🌓 **响应式布局**：适配不同屏幕尺寸
- 🖱️ **拖拽支持**：支持拖拽选择目录
- 📋 **日志显示**：实时显示操作日志，支持不同级别的日志信息
- ⏳ **加载状态**：执行时显示加载状态，防止重复操作

## 📦 安装说明

### Windows
1. 下载 `Unfold_0.1.0_x64_en-US.msi` 或 `Unfold_0.1.0_x64-setup.exe`
2. 双击安装程序，按照提示完成安装
3. 启动应用：开始菜单 → Unfold

### macOS
1. 下载 `Unfold_0.1.0_aarch64.dmg`（Apple Silicon）或 `Unfold_0.1.0_x64.dmg`（Intel）
2. 打开 DMG 文件，将 Unfold 拖动到 Applications 文件夹
3. 启动应用：Launchpad → Unfold

### Linux
1. 下载 `Unfold_0.1.0_amd64.AppImage`
2. 添加执行权限：`chmod +x Unfold_0.1.0_amd64.AppImage`
3. 运行应用：`./Unfold_0.1.0_amd64.AppImage`

## 🚀 使用方法

### 快速开始
1. 启动 Unfold 应用
2. 选择源目录（包含嵌套文件的目录）
3. 选择目标目录（文件将要平铺到的目录）
4. 配置操作选项：
   - **移动/复制**：选择移动或复制文件
   - **演习模式**：勾选以预览操作结果
   - **冲突策略**：选择覆盖、跳过或重命名
   - **清理空文件夹**：勾选以清理空文件夹
5. 点击"开始执行"
6. 查看操作日志，确认结果

### 功能说明

#### 冲突处理策略
- **覆盖**：直接覆盖已存在的文件
- **跳过**：跳过已存在的文件，保留原文件
- **重命名**：自动重命名文件（如 `file.txt` → `file-1.txt`）

#### 撤销操作
- 只能撤销上一次的移动操作
- 复制操作和演习模式无法撤销
- 撤销后操作日志将被清除

#### 操作日志
- 查看所有操作历史
- 记录操作时间、源路径、目标路径、操作类型
- 支持导出日志

## 📝 已知问题

1. 演习模式下，撤销功能不可用
2. 大量文件（10000+）时，UI 可能会有短暂卡顿
3. 路径包含特殊字符时，可能出现意外错误
